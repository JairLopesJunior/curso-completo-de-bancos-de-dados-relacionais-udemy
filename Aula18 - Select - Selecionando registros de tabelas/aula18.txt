// Select - Selecionando registros de tabelas

SQL
  - Conjunto DDL (Data Definition Language)
    - CREATE, DROP

  - Conjunto DML (Data Manipulation Language)
    - INSERT, UPDATE, DELETE

  - Conjunto DQL (Data Query Language)
    - SELECT


// SELECT
Ex: 
- select * from aluno;               // Seleciona tudo.
- select nome, telefone from aluno;  // Seleciona somente o nome e o aluno.
