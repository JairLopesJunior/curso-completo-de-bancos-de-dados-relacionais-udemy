Extra - Criando e utilizando Sequences

O que é Sequence?
R: É basicamente um recursos que permite gerar numeros sequenciais, quando define uma coluna do tipo serial
automaticamente cria uma sequence para a coluna, sendo que esta sequence possui alguns parametros pre-definidos
como por exemplo ela será incrementada de 1 em 1 partindo do valor 1, porem pode-se criar sequences customizadas
e também é possivel compartilhar a sequence para tabelas diferentes.

banco de Dados:
Esquema:

Tabela1 <---------- Sequence
Tabela2 <---------- Sequence
Tabela3 <------------|


create sequence produto.sequence_x_1 increment 100 start 10000;
drop sequence produto.sequence_x_1;

create table produto.tb_teste_1(
	id_teste_1 int default nextval('produto.sequence_x_1') primary key,
	teste varchar(100)
);

insert into produto.tb_teste_1(teste)values('Teste 500');
select * from produto.tb_teste_1;
select * from produto.tb_teste_2;

drop table produto.tb_teste_1;
drop table produto.tb_teste_2;

create table produto.tb_teste_2(
	id_teste_2 int default nextval('produto.sequence_x_1') primary key,
	teste varchar(100)
);
